Hi folks;

This is a quick command overview of all the commands the LAURA II engine and its supportive engine GALE provide to create or modify games in the LAURA II engine.
You cannot use pre-compiled Lua scripts, you'll have to provide Lua source only (LAURA II (or rather GALE) just doesn't support it).

Now I take it that you know how Lua works. So I won't go into the deep of that, and if you don't then just google around for some good tutorials about Lua. Here I only list the stuff that LAURA II and GALE add to Lua in order to enable you to create an RPG or Zelda-inspired adventure in LAURA II

Image.Assign

Overview
NameImage.Assign
TypeMethod
SyntaxImage.Assign(Image,NewImageName,KeepOriginal=0)

Assigns an image to a new name so that it can be used in an easier way with other scripts.

function Main()
 img = Image.Load("MyImage.png")
 Image.Assign(img,"MyImage")
 Image.Show("MyImage",10,10)
end
Of course, the code above can also be made shorter. Like this:
function Main()
 Image.Assign(Image.Load("MyImage.png"),"MyImage")
 Image.Draw("MyImage",10,10)
end
Notes
  • You don't have to release the original image code. That happens automatically unless you set the KeepOriginal parameter.
  • If an image is already on that spot it's automatically removed. You do not have to free it first.
  • The strings you use for this feature are CASE SENSITIVE!
